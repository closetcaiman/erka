# Drill

`JR.txt` contains closed questions transcribed from the UPeL quiz PDFs in `materials/quiz-upel/`.

Questions are in Polish (same language as the source PDFs).

**Drill app:** https://gronostajo.github.io/drill2/

## Format

```
Question text
A) option
>>>B) correct option
C) option
```

`>>>` marks the correct answer. Multiple `>>>` lines are allowed for multi-select questions.
